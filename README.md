#  Weather App

A simple Weather App built using HTML, CSS, and JavaScript.

## Features

- Search weather by city name
- Shows current temperature
- Displays humidity
- Displays wind speed
- Dynamic weather icons
- Error message for invalid city names
- Responsive and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

## How It Works

1. User enters a city name.
2. The app sends a request to the OpenWeather API.
3. Weather data is fetched.
4. Temperature, humidity, wind speed, and weather condition are displayed.
5. Weather icon changes according to the current weather.

## API Used

OpenWeather API
